# 💳 Credit Card Transaction & Customer Analysis (SQL + Power BI)

## 📊 Project Overview
This project focuses on analyzing credit card transaction data and customer behavior using SQL and Power BI.  
It follows an end-to-end workflow from data loading and cleaning in SQL to building interactive dashboards in Power BI.

---

## 🔄 Project Workflow
1. Loaded raw CSV data into SQL database  
2. Performed data cleaning and transformation using SQL  
3. Connected SQL database to Power BI  
4. Built interactive dashboards for insights
